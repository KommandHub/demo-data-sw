# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.3] - 2026-07-04

### Added
- Added support for PHP 8.1 and above.
- Added PHP 8.1, 8.3, and 8.4 to CI pipeline.

## [1.0.2] - 2026-07-04

### Added
- Documentation for the `demo-` image naming convention in README.

### Changed
- Refined media filtering to include all images starting with `demo-` instead of just `demo-product`.

## [1.0.1] - 2026-07-04

### Changed
- Expanded Shopware support to include version 6.6.0.

## [1.0.0] - 2026-06-06

### Added
- Initial release of the Shopware Plugin Boilerplate.
- Pre-configured PHPUnit, PHPStan, and PHP-CS-Fixer.
- Docker-based development environment.
- Standard open-source documentation (README, CONTRIBUTING, LICENSE, CODE_OF_CONDUCT).
- Makefile for common development tasks.
